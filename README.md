# DEX

The aim of the project is to create a DEX for swap tokens, including fees, liquidy pools and staking

## Description

### Frontend
    * React
    * Typescript

### Backend
    * Go
    * PostgreSQL

### CI/CD

#### CI
    * Dockerfile
    * Docker compose yaml file
    * Github Action on push: Build images and push them on private registry
    * Unit Tests
    * Integration Tests

#### Deployment
    * Vercel or Netlify
    * Cloud DB
    * Sitemaps
